# RhythmBridge

Frontend-only prototype (hackathon build) for RhythmBridge — making dance accessible through visual rhythm.

No backend, authentication, or API calls are implemented. All data is placeholder/sample data, and video is a sample clip for demo purposes.

## Stack
- React + Vite
- Tailwind CSS
- React Router
- Framer Motion
- lucide-react (icons)

## Run locally
```bash
npm install
npm run dev
```

## Pages
- `/` — Login (email/password UI, guest access, forgot password UI-only)
- `/dashboard` — Home dashboard with feature cards, recent activity, accessibility tips
- `/training` — Training Studio: rhythm lane, video player, lyrics + beat counter
